# About OpenGuild

OpenGuild is a builder-driven community centered around Polkadot. OpenGuild is built by Web3 builders for Web3 builders. Our primary aim is to cater to developers seeking a comprehensive understanding of the Polkadot blockchain, providing curated, in-depth materials with a low-level approach.

- **About us:** [Learn more about us](https://openguild.wtf/about)
- **Website:** [OpenGuild Website](https://openguild.wtf/)
- **Github:** [OpenGuild Labs](https://github.com/openguild-labs)
- **Discord**: [Openguild Discord Channel](https://discord.gg/bcjMzxqtD7)

## Platform Features

- [ ] On-chain NFT membership system hosted on Moonbeam Network
- [ ] Reward points, experience & ranking system
- [ ] Treasury management functionalities
- [ ] Treasury is driven by Polkadot OpenGov-like governance system
- [ ] Community NFT marketplace
- [ ] View & manage Github community issues
- [ ] View & manage Job board
